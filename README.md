# Portfolio Website

Welcome to my Portfolio Website repository! This simple website showcases my work, skills, and background using HTML, CSS, and a touch of JavaScript for day and night mode functionality.

## Features

- **Responsive Design**: The website is designed to adapt to various screen sizes, ensuring a seamless experience across devices.
- **Portfolio Showcase**: Showcases my projects, work experience, education, and skills.
- **Day and Night Mode**: Includes a toggle switch to switch between light and dark themes, providing a comfortable viewing experience for users.

## Usage

1. Clone the repository to your local machine:

```bash
git clone https://github.com/Anonymous096/portfolio.git
```


2. Open the `index.html` file in your web browser.

3. Explore different sections of the website to learn more about me, my projects, skills, and experiences.

4. Use the day and night mode toggle switch to switch between light and dark themes based on your preference.

## Customization

Feel free to customize the content, design, and functionality of the website to better reflect your personality and showcase your work effectively.
